# MariekeVDA.github.io

A small demo documentation site built with [GitHub Pages](https://pages.github.com/) and [Jekyll](https://jekyllrb.com/).

🌐 **Live site:** https://MariekeVDA.github.io

## Pages

- **Home** – `index.md`
- **Getting Started** – `docs/getting-started.md`
- **Installation** – `docs/installation.md`
- **Usage** – `docs/usage.md`

## Running Locally

```bash
bundle install
bundle exec jekyll serve
```

Then open http://localhost:4000.
